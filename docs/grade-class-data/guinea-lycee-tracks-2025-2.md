# Système Éducatif Guinéen : Programme du Lycée (2025–2026)

Ce document détaille le programme des trois années du lycée (11ème, 12ème, Terminale) pour les trois options (profils).

## 1. Les Trois Profils (Options)
* **Sciences Mathématiques (SM)** : Accent sur l'abstraction, l'algèbre et la physique.
* **Sciences Expérimentales (SE)** : Accent sur le vivant, la nature et la chimie.
* **Sciences Sociales (SS)** : Accent sur les humanités, l'analyse sociétale et l'économie.

---

## 2. 11ème & 12ème Année : Le Cycle de Base
Durant ces deux années, les élèves consolident les bases spécifiques à leur série.

| Caractéristique | Sciences Mathématiques (SM) | Sciences Expérimentales (SE) | Sciences Sociales (SS) |
| :--- | :--- | :--- | :--- |
| **Matières Principales** | **Mathématiques** (Algèbre/Analyse)<br>**Physique** (Mécanique/Optique)<br>Chimie | **Biologie & Géologie** (SVT)<br>**Chimie**<br>**Physique** | **Géographie**<br>**Histoire**<br>**Français** |
| **Matières Secondaires** | Français<br>Anglais<br>Biologie (Base)<br>EPS | Mathématiques (Appliquées)<br>Français<br>Anglais<br>EPS | Mathématiques (Base)<br>Anglais<br>Économie<br>EPS |

---

## 3. Terminale : Préparation au Baccalauréat
En classe de Terminale, la **Philosophie** devient obligatoire pour toutes les options. Le succès au Bac dépend des coefficients ci-dessous.

### Tableau des Coefficients (Estimatif)

| Matière | Profil SM | Profil SE | Profil SS |
| :--- | :--- | :--- | :--- |
| **Mathématiques** | **Coefficient le plus élevé** | Coefficient Élevé | Coefficient Faible (2) |
| **Physique** | **Coefficient Élevé** | Coefficient Élevé | - |
| **Chimie** | Coefficient Moyen | **Coefficient Élevé** | - |
| **Biologie / Géologie** | - | **Coefficient le plus élevé** | - |
| **Histoire** | Coefficient Faible | Coefficient Faible | **Coefficient le plus élevé** |
| **Géographie** | Coefficient Faible | Coefficient Faible | **Coefficient le plus élevé** |
| **Philosophie** | Coefficient Moyen | Coefficient Moyen | **Coefficient Élevé** |
| **Économie** | - | - | Coefficient Élevé |
| **Français** | Coefficient Moyen | Coefficient Moyen | Coefficient Élevé |
| **Anglais** | Coefficient Faible | Coefficient Faible | Coefficient Moyen |

> **Note Importante** : L'Économie est une matière critique exclusive au profil Sciences Sociales en Terminale, couvrant la mondialisation et les systèmes financiers.
