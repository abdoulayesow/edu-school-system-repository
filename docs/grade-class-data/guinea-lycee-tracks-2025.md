# Guinea Lycée Curriculum - Specialized Tracks (2025-2026)

## Overview

The Guinean Lycée (upper secondary) system was reformed starting in 2021. The high school cycle covers **11ème année**, **12ème année**, and **Terminale**, culminating in the **Baccalauréat Unique (BU)**.

This document details the three main specialized tracks:
- **SM** - Sciences Mathématiques
- **SE** - Sciences Expérimentales
- **SS** - Sciences Sociales

---

## System Structure

### Year Progression

| Year | Age | Structure | Notes |
|------|-----|-----------|-------|
| 11ème Année | 16-17 | Tronc Commun (2 series) | Scientific or Literary orientation |
| 12ème Année | 17-18 | Specialization begins | 4 tracks available |
| Terminale | 18-19 | Final specialization | 5 tracks, Baccalauréat exam |

### Track Evolution

```
11ème Année
├── Série Scientifique ──────┬── 12ème SM ──── Terminale SM (Bac C)
│                            │                 Terminale SE (Bac B2)
│                            └── 12ème SE ──── Terminale SE (Bac D)
│
└── Série Littéraire ────────┬── 12ème SS ──── Terminale SS (Bac B1)
                             └── 12ème LL ──── Terminale LL (Bac A)
```

## Franco-Arab Variants

In addition to the main tracks, Guinea offers Franco-Arab variants:

| Track | Code | Description |
|-------|------|-------------|
| Sciences Sociales Franco-Arabe | SS-FA | SS curriculum with Arabic language instruction |
| Sciences Expérimentales Franco-Arabe | SE-FA | SE curriculum with Arabic language instruction |

These tracks include **Arabic Language** as an additional core subject.

## Implementation Notes for GSPN System

### Database Considerations

1. **Track Codes**: Use standardized codes (`SM`, `SE`, `SS`, `LL`, `SEC`)
2. **Year Levels**: Map to existing grade system:
   - 11ème = index 11
   - 12ème = index 12
   - Terminale = index 13
3. **Subject Coefficients**: Store per track and per year
4. **Franco-Arab Variants**: Track with suffix (`-FA`)

### Suggested Schema Extension

```sql
-- Track/Series master table
CREATE TABLE lycee_tracks (
  id UUID PRIMARY KEY,
  code VARCHAR(10) UNIQUE NOT NULL, -- SM, SE, SS, LL, SEC
  name_fr VARCHAR(100) NOT NULL,
  name_en VARCHAR(100) NOT NULL,
  bac_type VARCHAR(10), -- A, B1, B2, C, D
  is_franco_arab BOOLEAN DEFAULT FALSE
);

-- Track-specific subject coefficients
CREATE TABLE track_subject_coefficients (
  id UUID PRIMARY KEY,
  track_id UUID REFERENCES lycee_tracks(id),
  subject_id UUID REFERENCES subjects(id),
  year_level INTEGER NOT NULL, -- 11, 12, 13
  coefficient INTEGER NOT NULL,
  is_specialized BOOLEAN DEFAULT FALSE
);
```
